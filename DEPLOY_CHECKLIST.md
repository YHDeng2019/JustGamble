## 🚀 Vercel 部署步骤（需要你操作）

### 1. 访问 Vercel
- [ ] 打开 https://vercel.com/login
- [ ] 用 GitHub 账号登录

### 2. 导入项目
- [ ] 点击 "Add New..." → "Project"
- [ ] 找到 `YHDeng2019/JustGamble` 仓库
- [ ] 点击 "Import"

### 3. 配置环境变量
打开 `ENV_VARS.md` 文件，按照说明添加 10 个环境变量：

#### AI API 配置（3个）
- [ ] `VITE_DEFAULT_API_BASE_URL`
- [ ] `VITE_DEFAULT_API_KEY`
- [ ] `VITE_DEFAULT_AI_MODEL`

#### Firebase 配置（7个）
- [ ] `VITE_FIREBASE_API_KEY`
- [ ] `VITE_FIREBASE_AUTH_DOMAIN`
- [ ] `VITE_FIREBASE_DATABASE_URL`
- [ ] `VITE_FIREBASE_PROJECT_ID`
- [ ] `VITE_FIREBASE_STORAGE_BUCKET`
- [ ] `VITE_FIREBASE_MESSAGING_SENDER_ID`
- [ ] `VITE_FIREBASE_APP_ID`

### 4. 开始部署
- [ ] 确认所有环境变量已添加
- [ ] 点击 "Deploy" 按钮
- [ ] 等待 1-2 分钟

### 5. 获取网址
- [ ] 部署成功后复制网址（例如：`https://just-gamble-xxx.vercel.app`）

## 📞 遇到问题？

如果部署失败或功能异常，检查：

1. **构建失败**
   - 查看 Vercel 构建日志
   - 确认 `package.json` 正确

2. **白屏**
   - 按 F12 打开控制台
   - 查看错误信息
   - 确认环境变量都已添加

3. **联机无法连接**
   - 确认 Firebase 环境变量正确
   - 检查 Firebase Realtime Database 规则

4. **AI 不决策**
   - 查看浏览器控制台错误
   - 确认 AI API 配置正确
