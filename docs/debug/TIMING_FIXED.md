# 联机模式时间配置已完全对齐单人模式

## 已修复的时间配置

### ✅ AI 思考时间
- **配置**：700ms + 1600ms × 纠结度 × 下注压力 + 随机400ms
- **范围**：700-2700ms
- **逻辑**：基于牌力强度（0-1），中等牌力最纠结，面对下注压力额外延长

### ✅ 获胜 Toast 显示时间
- **摊牌后停留**：6500ms
- **弃牌结束停留**：3500ms

### ✅ ActionBar 显示延迟
- **延迟**：600ms
- **目的**：避免连续决策时无间隔

### ✅ All-in 戏剧效果
- **持续时间**：1400ms（已从 2000ms 修正）

### ✅ 阶段切换停顿
- **延迟**：800ms
- **目的**：让玩家看清公共牌（FLOP/TURN/RIVER）

### ✅ 新手牌开始延迟
- **回合结束后等待**：8000ms（确保玩家看完 toast）
- **startNewHand 后延迟**：600ms
- **对齐单人模式**：600ms 桌面过渡 + 600ms dealInitialCards

### ✅ 快速摊牌（所有人 All-in）
- **每阶段间隔**：400ms
- **逻辑**：检测所有活跃玩家都 All-in 时触发快速摊牌
- **新增方法**：`quickDealToShowdown()`
