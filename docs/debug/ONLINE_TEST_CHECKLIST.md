## 📋 测试流程

### 准备工作
1. 打开两个不同的浏览器窗口（或使用 Chrome + Safari）
2. 每个窗口选择不同的用户账号

### 基础流程测试

#### 1. 创建和加入房间
- [ ] 窗口A：创建房间（设置2-4人，初始筹码1000，盲注10/20）
- [ ] 验证：房间码正确显示
- [ ] 窗口B：输入房间码加入
- [ ] 验证：两个窗口都能看到对方在房间中

#### 2. 添加机器人（可选）
- [ ] 房主：点击"添加机器人"
- [ ] 验证：机器人出现在玩家列表，带🤖标记

#### 3. 准备和开始
- [ ] 窗口B：点击"准备"，状态变为"✅ 已准备"
- [ ] 窗口A（房主）：所有玩家准备后，点击"🎮 开始游戏"
- [ ] 验证：两个窗口都进入游戏界面

### 游戏进行测试

#### 4. 发牌阶段
- [ ] 验证：两个窗口都能看到游戏界面
- [ ] 验证：玩家能看到自己的2张手牌
- [ ] 验证：看不到对手的手牌（显示为牌背）
- [ ] 验证：公共牌区域为空
- [ ] 验证：底池显示30（盲注之和）

#### 5. 回合提示
- [ ] 验证：轮到自己时显示"🎯 轮到你了！"（带脉冲动画）
- [ ] 验证：轮到对手时显示"⏳ 等待 XXX 行动..."

#### 6. 玩家操作
- [ ] 测试：弃牌（Fold）
  - 点击弃牌
  - 验证：自己的牌变暗/隐藏
  - 验证：回合切换到下一个玩家
  - 验证：对手窗口立即收到更新

- [ ] 测试：过牌（Check）
  - 当可以过牌时点击
  - 验证：无需下注
  - 验证：回合切换

- [ ] 测试：跟注（Call）
  - 点击跟注
  - 验证：筹码减少
  - 验证：底池增加
  - 验证：对手窗口同步更新

- [ ] 测试：加注（Raise）
  - 拖动加注滑条或输入金额
  - 点击加注
  - 验证：筹码减少正确数额
  - 验证：底池增加
  - 验证：对手需要再次行动

#### 7. 阶段推进
- [ ] PRE_FLOP 下注回合结束
  - 验证：自动发3张 flop 牌
  - 验证：两个窗口看到相同的公共牌
  - 验证：阶段提示"翻牌"出现

- [ ] FLOP 下注回合结束
  - 验证：自动发1张 turn 牌
  - 验证：阶段提示"转牌"出现

- [ ] TURN 下注回合结束
  - 验证：自动发1张 river 牌
  - 验证：阶段提示"河牌"出现

- [ ] RIVER 下注回合结束
  - 验证：进入摊牌阶段
  - 验证：所有玩家手牌显示
  - 验证：底池分配正确
  - 验证：赢家筹码增加

#### 8. 新一手开始
- [ ] 验证：摊牌结果显示3秒后
- [ ] 验证：自动开始新一手
- [ ] 验证：庄家位置轮转
- [ ] 验证：局数增加（第2局）
- [ ] 验证：手牌重新发放

### AI 机器人测试

#### 9. AI 决策
- [ ] 验证：AI 在1-2.5秒内自动行动
- [ ] 验证：AI 行动在日志中显示
- [ ] 验证：AI 的决策合理（不会无脑 all-in）

### 超时和断线测试

#### 10. 超时处理
- [ ] 测试：30秒不操作
- [ ] 验证：自动弃牌
- [ ] 验证：游戏日志显示"(超时)"
- [ ] 验证：回合继续进行

#### 11. 连接状态
- [ ] 测试：关闭网络或Firebase
- [ ] 验证：5秒后显示"🔄 正在重新连接..."
- [ ] 验证：10秒后显示"⚠️ 连接已断开"
- [ ] 测试：恢复网络
- [ ] 验证：连接状态恢复正常

### 退出和离开测试

#### 12. 中途退出
- [ ] 测试：游戏中点击"退出"
- [ ] 验证：弹出确认对话框
- [ ] 验证：确认后返回大厅
- [ ] 验证：另一个窗口游戏继续（如果还有其他玩家）

#### 13. 房主离开
- [ ] 测试：房主离开房间
- [ ] 验证：房主权限转移给下一个玩家
- [ ] 验证：游戏继续进行
